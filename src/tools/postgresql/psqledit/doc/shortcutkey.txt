
ショートカットキー((*)はSQLエディタでのみ有効)

 Ctrl + R   : SQL実行
 Ctrl + E   : Explain plan実行
 Ctrl + D   : 検索を実行し，結果をファイルに保存
 Ctrl + L   : SQL実行履歴を表示

 Esc        : キーワード補完(*)
 Shift + Esc: キーワード補完(逆順)(*)

 Ctrl + A   : 全て選択
 Ctrl + C   : 選択範囲をコピー
 Ctrl + X   : カット(*)
 Ctrl + V   : ペースト(*)
 Ctrl + Z   : Undo(*)
 Ctrl + Y   : Redo(*)

 Ctrl + F   : 文字列検索
 F3         : 次を検索
 Shift + F3 : 前を検索
 Ctrl + H   : 文字列置換(*)
 Alt + F3   : 検索文字列の強調表示を解除

 Ctrl + F2  : ブックマーク設定／解除(*)
 F2         : 次のブックマークへ移動(*)
 Shift + F2 : 前のブックマークへ移動(*)
 Ctrl+Shift+F2: ファイル内のすべてのブックマークを解除(*)

 Ctrl + W   : 実行結果欄のビューを切り替える(テキスト形式，表形式)
 Ctrl + Q   : SQLエディタと実行結果欄を移動する

 Ctrl + N   : 新規ウィンドウを開く
 Ctrl + O   : ファイルを開く
 Ctrl + S   : SQLをファイルに保存
 Ctrl + F6  : 次のウィンドウに移動
 Ctrl + F4  : ウィンドウを閉じる
 Alt + F4   : アプリケーションを終了

 Tab        : 選択範囲の各行に，Tabを挿入する(*)
 Shift + Tab: 選択範囲の各行から，Tabを削除する(*)
 Ctrl+Shift+U: 選択中の領域を大文字に変換する(*)
 Ctrl + U   : 選択中の領域を小文字に変換する(*)

 Ctrl + P   : エディタのSQLを印刷する(*)
